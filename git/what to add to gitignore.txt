A file should be ignored if any of the following is true:
	- The file is not user by your project
	- the file is not used by anyone else in your team
	- the file is generated by another process


Now, lets take a look at common types of files we would ignore:
	- operating system files
		- thumbs.db
		- .DS_Store
	- Application files
		- files generated by IDEs and editors
	- language and framework files
		- ejm: npm-debug.log
	- files, downloaded with package managers
		- ejm: bower_components, node_modules
	- credentials
		- ejm: wp-config.php
