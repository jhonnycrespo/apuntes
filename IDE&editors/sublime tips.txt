LINE AND IDENTATION TOOLS
===========================
   LINE

	* ctrl+shift+arrow: permite mover un texto hacia arriba o hacia abajo
	* ctrl+shift+d: copiar un texto. es necesario tener el cursor dentro el texto.
	* ctrl+shift+k: eliminar un texto.

   IDENTATION

	* ctrl + [ or ]: identar un texto a la izquierda o derecha.
	* edit -> line -> Reindent: seleccionar todo un texto para reidentar.
	* ctrl + shift + v: pegar e identar. para que no se desconfigure al pegar con ctrl+v.

GOTO ANYTHING
=============
	* ctrl + p: buscar y abrir un archivo. 
		-En el cuadro de texto que se abre tambien podemos buscar una funcion en el
		    archivo agregado @<nombreFuncion>.
		- ":" para buscar un numero de linea;
		- "#" para buscar una cadena.

	* Podemos usar directamente los comandos de busqueda anteriores si ya estamos en el archivo .

COMMAND PALETTE
================

	* ctrl+shift+p: abrir ventana de comandos.
		- convertir una seleccion a mayuscula
		- sort lines: cambiar orden de una seleccion
		- Encode Special Characters: convierte cadenas como {"", &} en html.
		- set sintax: cambiar sintaxis del archivo.
